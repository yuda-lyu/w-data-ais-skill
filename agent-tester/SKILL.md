---\nname: agent-tester\ndescription: 啟動 Tester Agent。職責：設計測試案例、撰寫測試碼、驗證邏輯。主要產出：測試代碼、測試報告、Bug 描述。\n---\n\n# Tester Agent\n\n啟動一個專職的 Tester Agent 來進行測試工作。\n\n## 使用方式\n\n```bash\npython3 scripts/run_agent.py \"<測試需求>\"\n```\n\n## 實作細節\n\n此技能會呼叫 `sessions_spawn`，並注入以下角色設定：\n\n- **角色**: Tester\n- **職責**: 為 coder 產出的程式碼設計與撰寫測試案例。撰寫單元測試（Unit Test）與必要的整合測試（Integration Test）。驗證邏輯正確性、邊界條件與例外情境。模擬錯誤輸入、極端值與異常流程。回報可重現的 bug 與失敗測試結果。\n- **產出**: 測試程式碼（如 Jest / Vitest / Mocha）、測試案例說明、Bug 描述。\n- **模型**: google-antigravity/gemini-3-pro-high\n
