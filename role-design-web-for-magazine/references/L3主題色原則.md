# L3 主題色原則：2 種明暗基底 + 5 套視覺語氣

## 色彩總規則

一個作品只採用一組主題色。亮色與暗色是同一主題的兩個表面，不是兩套不同設計。

基本 token：

```css
--ink: 深色文字與暗底;
--ink-rgb: 深色 RGB;
--paper: 亮色紙面;
--paper-rgb: 亮色 RGB;
--paper-tint: 紙面次層;
--ink-tint: 暗色次層;
--accent: 少量焦點色;
```

使用規則：

- 文字、線條、icon 優先使用 `currentColor`，讓元件能在 light / dark 表面自動適配。
- 大面積背景只用 `paper` 或 `ink` 系列；accent 不作大面積底色。
- 不接受任意 hex 混搭；若有品牌色，先映射到 token，再檢查對比。
- 同一頁內最多 1 個主 accent；資料視覺化除外，但顏色必須承載語意。

## 1. 亮色紙面基底

**角色**：承載長時間閱讀、資料、表格、截圖、產品說明、案例細節。

**性格**：清楚、可信、開放、有紙本呼吸感。

**安排**：

- 背景用 `paper`，文字用 `ink`。
- 區塊之間用細線、留白、欄寬區分，不依賴陰影。
- 圖片與截圖優先放在亮色基底，資訊辨識度最高。

## 2. 深色背景基底

**角色**：承載首頁 hero、章節開場、金句、問題頁、結尾、沉浸式展示。

**性格**：有儀式感、凝聚注意力，承載少字與強觀點。

**安排**：

- 背景用 `ink`，文字用 `paper`。
- 可讓影像、材質或靜態背景在深色底下稍微透出。
- 暗色段落文字量要少；正文密集時回到亮色紙面。

## 3. 黑色與暖白

**性格**：最接近傳統雜誌基底，以接近黑色的深色與暖白紙色形成穩定對比。

```css
--ink:#0a0a0b;
--ink-rgb:10,10,11;
--paper:#f1efea;
--paper-rgb:241,239,234;
--paper-tint:#e8e5de;
--ink-tint:#18181a;
--accent:#b88a3d;
```

**避免**：加入鮮豔品牌色或大量彩色 icon，會破壞黑白紙面感。

## 4. 深藍與冷白

**性格**：冷靜、理性、學術期刊感，以深藍色與冷白色形成清晰秩序。

```css
--ink:#0a1f3d;
--ink-rgb:10,31,61;
--paper:#f1f3f5;
--paper-rgb:241,243,245;
--paper-tint:#e4e8ec;
--ink-tint:#152a4a;
--accent:#5f7fa8;
```

**避免**：把整頁做成高飽和藍色介面；藍只應是深色基底的偏向，不是裝飾主題。

## 5. 深綠與象牙白

**性格**：沉穩、有呼吸感，以低飽和深綠色搭配象牙白紙色。

```css
--ink:#1a2e1f;
--ink-rgb:26,46,31;
--paper:#f5f1e8;
--paper-rgb:245,241,232;
--paper-tint:#ece7da;
--ink-tint:#253d2c;
--accent:#7f9164;
```

**避免**：搭配過多植物插畫或亮綠，會變成環保宣傳模板。

## 6. 深棕與暖米

**性格**：溫暖、有年代感，以深棕色搭配暖米色紙面。

```css
--ink:#2a1e13;
--ink-rgb:42,30,19;
--paper:#eedfc7;
--paper-rgb:238,223,199;
--paper-tint:#e0d0b6;
--ink-tint:#3a2a1d;
--accent:#9b6b35;
```

**避免**：整頁棕色過重。要用足留白、黑白圖片或細線平衡。

## 7. 炭灰與沙米色

**性格**：炭灰與低飽和沙米色，克制、中性、安靜。

```css
--ink:#1f1a14;
--ink-rgb:31,26,20;
--paper:#f0e6d2;
--paper-rgb:240,230,210;
--paper-tint:#e3d7bf;
--ink-tint:#2d2620;
--accent:#b89b6f;
```

**避免**：大面積米色但缺少黑色結構，會顯得軟弱。必須用強字級、深墨線條、真實圖片支撐。

## 主題選擇快速規則

| 視覺需求 | 推薦 |
|---|---|
| 最穩定、最接近雜誌基底 | 黑色與暖白 |
| 冷靜、理性、深色偏藍 | 深藍與冷白 |
| 沉穩、自然、低飽和深綠 | 深綠與象牙白 |
| 溫暖、紙張感、復古但不厚重 | 深棕與暖米 |
| 克制、中性、安靜 | 炭灰與沙米色 |
| 長時間閱讀 | 任一主題的亮色紙面 |
| 強記憶點或章節停頓 | 任一主題的深色背景 |

## 配色 × 字體搭配參考

對應上方五套主題色，依 L2 #1 三分工（襯線標題 + 非襯線正文 + 等寬元資料）：

| 主題 | ink / paper | 推薦字體（標題 + 正文 + 等寬） | 視覺語氣 |

## 衍生色階建議

若需自行衍生色階（例如 paper-tint、ink-tint 中間值或 accent 變體），建議用 oklch 而非 hex/rgb——oklch 在亮度維度上感知均勻，色階過渡更自然。
